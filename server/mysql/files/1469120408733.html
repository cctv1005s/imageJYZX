教学目标
 
详细说明学生学习课程后在知识、技能、态度等方面达到的状态，陈述应力求明确、具体，并可以观察和测量，600字以内
 
随着计算机技术的飞速发展与社会信息化的强烈需求，我国对高校毕业生的计算机应用与软件编程水平的要求日益提高，面向对象与可视化编程技术已成为高等院校计算机相关专业学生应该掌握的基本技能。本课程以流行的可视化编程工具Visual C++为平台，从面向对象的基本概念出发，讲述可视化程序设计的基本思想与方法。通过本课程的课堂讲授与课后练习，力求让学生能够掌握面向对象思想与可视化编程方法，使用Visual C++平台设计各种单机及网络应用程序，并能够从事系统软件的开发工作。
 
课程在学生培养中的地位和作用
 
课程开设的必要性及其在教学计划中对学生培养的作用，400字以内
 
随着面向对象编程技术的兴起，各种面向对象的可视化编程工具出现，它打破了已有的模块化编程模式，以全新的思想、高效的性能和良好的用户界面受到计算机界人士的普遍欢迎。为了使学生能够掌握该项技术，特开设这门选修课，重点介绍面向对象的可视化编程技术及其相应工具。由于本科低年级已讲授C++语言，因此本课程以Visual C++为平台，介绍基于MFC的Windows程序开发技术，内容包括：Windows图形界面应用程序、ActiveX控件、数据库程序和网络应用程序等。本课程具有很强的实践背景，强调学生的实际动手能力，是对学生软件开发综合能力的培养，因此在教学计划中起着比较重要的作用。

 

 
主要教学手段和方法
为完成教学目标而采用的主要教学方法和手段，以及方法和手段的改革情况，600字以内
 
本课程采用的Visual C++是目前广泛使用的可视化开发环境，其采用的面向对象的编程方法是当前软件开发的重要手段之一。为了满足计算机科学与技术等专业本科生的软件编程能力培养的需求，本课程采用将课堂教学与课后练习相结合的教学方式。
课堂教学采用多媒体投影方式讲授可视化编程的基础知识，通过不同编程实例的现场演示来介绍不同知识点的操作方法，将可视化编程的基础知识与操作方法紧密结合起来，使学生能够掌握可视化编程的基本方法。同时，本课程在Windows图形界面应用程序设计的基础上，介绍了数据库应用程序与网络应用程序的设计与实现方法，符合当前多数应用程序运行在网络环境下并依赖数据库支持的特点。
课后练习采用编程作业的方式完成，针对重点章节设计了多个相关的编程题目，由学生在上机课以及课后时间独立完成，以促进学生对面向对象与可视化编程相关知识的理解。

明确说明考试、平时成绩（讨论、作业、测验、出勤等）、实验实践所占总成绩比重，以及考试的形式（闭卷、开卷），400字以内

课程学习要求和建议
对学生学习该课程的相关要求及学习建议，800字以内
 
本课程的特点是从面向对象的基本概念出发，讲述可视化程序设计的基本思想与方法。本课程的课堂教学主要讲授可视化编程的基础知识，并通过编程实例的现场演示来介绍具体知识点的操作方法。由于本课程具有很强的实践性特点，仅依靠课堂教学无法保证学生有效掌握编程方法，因此课后练习是本课程教学中的重要环节与辅助手段，学生需要在上机课以及利用课后时间完成一定的编程作业，以加深学生对面向对象与可视化编程相关知识的理解。

2.教学环节的内容
课后习题：5组习题（分别对应不同章节），共计30学时
期末考试：1个综合性习题（涉及主要章节的主要知识点），用时3学时
3.课后习题的内容
习题1：
  • 设计学生类(Student)，数据成员是学号、姓名与成绩(数学、外语与计算机)，对类对象数组进行输入；设计以学号为参数的搜索函数，从数组中搜索并返回学生信息
  • 定义一个二维方阵类，通过重载二元运算符“+、´”，求二维方阵的加法和乘法
习题2：
  • 编程实现键盘鼠标测试程序，满足以下要求：按下键盘任意键，屏幕显示按键信息；单击鼠标左键，屏幕显示鼠标信息；假设鼠标右键失灵，用Ctrl+鼠标左键代替；自定义消息，带两个参数，由“?”键激活，屏幕显示相应信息
  • 编程实现SmallBall程序
习题3：
  • 设计单文档窗口程序，用户通过菜单打开一个对话框，在其中编辑框控件中输入文本，并单击“确定”按钮，在客户区输出相应文本
  • 设计单文档窗口程序，带有浮动菜单，用户选择某个菜单项时，在客户区输出相应图形，并可将图形类型与坐标保存到文档中
习题4：
  • 设计一个单文档边框窗口程序，要求具有以下几个功能：带快捷菜单；通过点击菜单项输出相应的图形；通过工具栏按钮保存与打开图形；按键盘上的“M”键开始循环操作，使一个位图(自行设计)沿着图形移动
习题5：
  • 设计一个单文档边框窗口程序，要求至少具有以下功能：程序具有打开对话框的功能；对话框中至少包含8种不同控件，实现个人信息的填写功能；通过确定按钮，在视图中显示个人信息；通过工具栏按钮，保存与打开个人信息
 
 
课程简要介绍
简要介绍课程的目标、主要授课内容、授课对象以及在学生培养中的作用，200—500字
 
随着面向对象编程技术的兴起，各种面向对象的可视化编程工具出现，它打破已有的模块化编程模式，以全新的思想、高效的性能和良好的用户界面受到计算机界人士的普遍欢迎。
本课程重点介绍面向对象的可视化编程技术及其相应工具。由于计算机科学与技术专业本科生已学习C++语言，因此本课程以流行的Visual C++编程工具作为授课基础。本课程讲授的内容主要包括：Windows编程基础（包括C++语言基础与Visual C++ 2010编程环境），MFC基础知识与基本编程方法，边框窗口、文档与视图，Windows图形接口与绘图，对话框、控件与控件栏，ActiveX控件，数据库应用程序，网络应用程序。
本课程具有很强的应用实践背景，强调培养学生的实际动手能力，通过将课堂讲授与上机练习相结合，注重学生的软件开发综合能力的培养。
英文课程简要介绍
课程介绍的英文翻译版
 
With the development of object-oriented technology, a variety of object-oriented visual programming tool rise, it breaks the modular programming mode existing. Because of the new idea, high efficiency and good user interface, it has been widely welcomed by computer professionals.
This course introduces the object-oriented visual programming technology and its corresponding tool. The undergraduate of computer science and technology have been learning C++ language, so the popular programming tool (Visual C++) was selected as the base of this course. The content of this course mainly includes: the windows programming basis (including the C++ language and Visual C++ 2010); the basic programming knowledge based MFC; frame window, view and document; windows graphic interface and drawing application; dialog, control and control bar; ActiveX control; database application software; network application software.
This course has very strong application background, emphasize the development of students’ practical ability, the classroom teaching combined with practice, and cultivate the comprehensive ability of students in software development.
