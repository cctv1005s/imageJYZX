本课程是为全校学生开设的一门公共选修课程，其主要包括三个部分：世界遗产概论、中国的世界遗产和其它各大洲的世界遗产。第一部分主要包括世界遗产概论、濒危世界遗产、候补和落选世界遗产；第二部分主要包括中国的世界文化遗产、中国的世界自然遗产、中国的双重世界遗产、中国的世界文化景观遗产和中国的世界非物质文化遗产；第三部分主要包括亚洲的世界遗产、大洋洲的世界遗产、非洲的世界遗产、欧洲的世界遗产和美洲的世界遗产等内容。

本课程试图通过教师的上课讲授、定量的视频播放和定期的课程报告等一系列教学环节使学生了解、熟悉和掌握世界遗产的内容、范围、标准和评价，特别是从《保护世界文化与自然遗产公约》和《保护非物质文化遗产公约》的基础上着重介绍、阐述和分析对世界遗产保护的理念。
